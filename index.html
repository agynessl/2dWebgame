Game Design

Author: Jinan Hu & Qiaoyu Liao

Overview:

XXX is a classic “Hop and Dodge” action game. It is quite similar to Varmintz Deluxe and Crossy Road. The player can use up, down, left and right keyboard inputs to control the main character to avoid enemies and to collect coins. The goal of the game is to achieve high score within a certain amount of time. When the time is up, no matter of whether the player finish the game or not, the game score will be calculated and save in the server. Moreover, the TOP 10 scores and the players’ names are displayed as a honor table.

Specifications:

  1. This game will be one-level endless encountered game. The game finishes when the character “died” or time has used up. The goal for the user is to go as far UP as possible and collect as many coins as possible.
  2. The basic design of the game will be based on the grid system of 10*25, the animal that the user control will take one square on the canvas. Each object like enemy or coin will also take one square.
  3. User uses the up, down, left, right arrow key to control the animal to move on the canvas.
  4. When the character encountered the enemy, falls into the river, or run over by a car, it is considered to be “dead”, hence the game ends.
  5. Or when the time specified(usually 60 seconds) is used up, the game ends.
  6. At the end of game, the score of this user get in this game will be calculated based on how far the character has moved in the map and how many coins it has collected. Each up distance is considered 1 point and each coins is considered 2 points.
  7. If the score user get is higher than the top ten score stored in the server, a box will be prompted to ask user’s name to store in the server.
